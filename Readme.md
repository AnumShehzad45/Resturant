## Project Summary
Nomnosh Sahiwal is a restaurant website designed to showcase the restaurant's offerings, location, and deals. The website features a modern, user-friendly interface with a focus on food presentation and easy navigation. It includes a responsive navbar with all current deals, a functional search bar, location information, and contact details in the footer.
